# 💰 Simple Interest Calculator

A responsive web application that calculates Simple Interest using HTML, Bootstrap, and JavaScript.

## 📖 Overview

This project allows users to calculate the interest earned on a principal amount over a specified number of years at a given interest rate.

The application demonstrates:

* HTML form creation
* Bootstrap UI components
* JavaScript DOM manipulation
* User input validation
* Financial calculations

---

## 🚀 Features

* Modern Bootstrap interface
* Responsive design
* Input validation
* Real-time calculation
* Beginner-friendly source code
* Clean and organized layout

---

## 🧮 Formula Used

Simple Interest is calculated using the following formula:

```text
SI = (Principal × Years × Rate) / 100
```

### Example

```text
Principal = $1000
Years     = 2
Rate      = 5%

SI = (1000 × 2 × 5) / 100
SI = $100
```
